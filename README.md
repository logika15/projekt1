📝 Технічне резюме проєкту: Python Quest 2026
Python Quest 2026 — це клієнт-серверний ігровий застосунок, що складається з набору мініігор. Проєкт демонструє роботу з мережевими протоколами, побудову графічних інтерфейсів (GUI) та реалізацію ігрової логіки.

🛠 Технологічний стек
Мова програмування: Python 3.x

Мережева взаємодія: Бібліотека socket (протокол TCP/IP).

Графічний інтерфейс: tkinter (стандартна бібліотека для GUI).

Робота з графікою: Pillow (PIL) для рендерингу сторонніх зображень.

Архітектура: Клієнт-серверна (один сервер — один клієнт).

🏗 Архітектура системи
Серверна частина (server.py):

Слухає вхідні з'єднання на localhost:12345.

Обробляє вхідні пакети даних у форматі рядків (наприклад, SCORE:33).

Зберігає стан загального рахунку гравця протягом сесії.

Надсилає підтвердження про отримання балів клієнту.

Клієнтська частина (client.py):

Модуль ініціалізації: Встановлює зв'язок із сокетом сервера.

Диспетчер екранів: Функція clear_screen очищує вікно перед завантаженням нового рівня, забезпечуючи плавну зміну інтерфейсу.

Ігрові рівні:

Level 1 (Візуальний): Аналіз зображення та обробка текстового вводу.

Level 2 (Логічний): Реалізація гри "Крестики-Нолики" з простим ШІ (ботом), що робить випадкові ходи.

Level 3 (Інтерактивний): 2D-лабіринт на базі компонента Canvas з обробкою подій клавіатури (KeyBinding).

🕹 Функціональні особливості
Стилізація (UI/UX): Використання єдиної кольорової палітри (Dark Theme) та кастомних стилів для кнопок і шрифтів.

Синхронізація даних: Кожна перемога на рівні автоматично надсилає дані на сервер, що гарантує збереження результатів.

Динамічне керування: Рівень лабіринту використовує систему координат для перевірки колізій (зіткнень) зі стінами та точкою виходу.

🚀 Можливості для масштабування
Багатокористувацький режим: Використання модуля threading на сервері для одночасної гри декількох клієнтів.

База даних: Інтеграція sqlite3 на стороні сервера для постійного зберігання таблиці лідерів.

Шифрування: Додавання модуля hashlib або ssl для захисту переданих даних.
